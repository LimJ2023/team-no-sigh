인터페이스(명세서,약속)
추상클래스(미완성 설계도)


추상메서드 : 일반메서드 사용 가능.
일반 메서드 o  멤버변수 o


인터페이스 : 모든 메서드가 추상 메서드
추상메서드 + 상수
인스턴스x 멤버변수x

자바8+에서
디폴트 메서드 // default
구현코드까지 작성하는 메서드
기본으로제공, 재정의 가능 

정적 메서드(static) : 인스턴스 생성 상관 없이 사용할 수 있는 메서드
(데이터 영역에 이미 할당되었기 떄문에)

private 메서드 : 자바9+에서 가능
인터페이스 구현한 클래스에서 공통으로 사용. 재정의 x

상속 exteds
인터페이스는 implements <- 구현의 책임 있음

차이점*

1.구현방식 
인터페이스 : 추상메서드 + 상수만 가능
추상클래스 : 일반메서드, 멤버변수 가능
2. 다중 상속
인터페이스 : 다중상속 가능 (여러개의 인터페이스 상속 가능)
추상클래스 : 다중상속 불가 (하나의 클래스만 상속받음)


인터페이스를 다중 상속받을 경우

접근할 수 없음
