상위 클래스가 추상클래스일경우 객체 생성 불가

하위 클래스에게 구현을 위임.

하위 클래스는 추상클래스의 구현 책임이 있다.

추상 메서드 -> 구현부가 없이 선언만 한 메서드.

abstract : 추상 선언

implement : 구현하다

구현부가 비어있다 <--> 구현부가 없다. :  차이 있음.


추상 클래스를 상속받은 추상클래스도 가능

추상 클래스를 구현할 경우 모든 추상 메서드를 구현해야한다.
(아닐시 추상클래스화 시킴)


템플릿 메서드 : 같은 순서의 메서드를 호출하는 메서드의 집합
(final 붙임)
