접근제어자 
public : 접근 제한 없음
protected : 같은 패키지 가능, 다른 패키지의 상속받은 클래스 가능
default : 같은 패키지 가능
private : 같은 클래스만 가능

멤버변수에 접근제어자를 붙여 접근할수있는 범위를 결정할수있다



get set : 객체의 데이터를 캡슐화하기 위해서.
외부에 직접 접근하지 않고 안전하게 값을 가져오기 위해서.


enum : 열거형 타입. 특정 값의 집합 정의.

일반적으로 상수 값의 집합을 표현함

예) 요일, 월, 색상 등 열거할때.
