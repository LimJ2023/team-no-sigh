생성자 (생성자 메소드)

생성자는 클래스 이름과 같다.


this. <- 자기 클래스를 가르킴

사용법 : 1.매개변수를 받아서 멤버필드를 초기화하는 경우. 2. 자신의 클래스 전체를 매개변수로 전달하는 경우.


이클립스 게터,세터 만드는법 : 

우클릭 -> source -> generate getter and setter 클릭

(alt + shift + s)



일반적으로 우리가 프로그래밍하여 만드는 
Class는 Static영역에 생성되고 
new 연산을 통해 생성한 객체는 Heap영역에 생성된다.

- static으로 선언한 변수는 0으로 초기화가 된다는 것.
- static 변수는 외부참조가 되지 않는다는 점. (= 정보은식)
- static 변수는 매개변수로 사용할 수 없다는 점.




get set : 객체의 데이터를 캡슐화하기 위해서.
외부에 직접 접근하지 않고 안전하게 값을 가져오기 위해서.
